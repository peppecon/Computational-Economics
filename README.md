# Computational Economics

This repository contains the computational files I developed and worked on during my PhD at Bocconi University. It includes implementations of various numerical methods and economic models focused on macroeconomic dynamics and computational finance.

## Contents

- **[Projection Methods with NGM Application](./Projection%20Methods%20with%20NGM%20Application)**: Implementation of Chebyshev projection methods applied to the Neoclassical Growth Model (NGM), including teaching materials, stochastic simulations, and performance comparisons.
